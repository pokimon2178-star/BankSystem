Bank System Project
Данный проект представляет собой систему управления банковскими счетами, реализованную на языке C#.
Основной функционал
Абстрактная логика: создан базовый класс AbstractAccount, включающий уникальный ID, данные владельца, защищенный баланс и статус (активный, замороженный, закрытый).
Расширенная реализация: класс BankAccount реализует автоматическую генерацию ID, поддержку различных валют (RUB, USD, EUR, KZT, CNY) и строгую валидацию всех операций.
Безопасность и валидация:
Запрет любых операций при неверном статусе счета (заморозка или закрытие).
Проверка корректности вводимых сумм и защита от отрицательных значений баланса.
Кастомная обработка ошибок: внедрена система специфических исключений для точной диагностики проблем:
AccountFrozenError — операция со счетом в статусе "Заморожен".
AccountClosedError — попытка доступа к закрытому счету.
InvalidOperationError — некорректный ввод данных.
InsufficientFundsError — попытка снятия суммы, превышающей баланс.
Формат вывода данных
При печати информации о счете отображаются:
Тип счета.
Имя владельца (клиента).
Маскированный ID (только последние 4 цифры).
Текущий статус.
Баланс с указанием валюты.
Тестирование
В проекте реализован сценарий тестирования, включающий создание счетов с разными статусами, валидные транзакции и проверку срабатывания защитных механизмов при попытке изменения замороженного счета.
