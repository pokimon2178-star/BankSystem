# BankSystem (Console Application)

Простая консольная система имитации банковского счета на C#, демонстрирующая принципы ООП (инкапсуляция, наследование, полиморфизм).

## 🛠 Архитектура проекта

Проект разделен на логические модули для удобства поддержки:

* **Enums.cs** — Типы валют (`RUB`, `USD`, `EUR` и др.) и статусы счета.
* **Exceptions.cs** — Кастомные исключения для обработки ошибок (заморозка, нехватка средств).
* **AbstractAccount.cs** — Базовый абстрактный класс, определяющий интерфейс счета.
* **BankAccount.cs** — Основная реализация счета с проверкой бизнес-логики.
* **Program.cs** — Точка входа и консольное меню управления.

## 🚀 Функционал

Приложение позволяет:
1. Создать счет в одной из 5 валют.
2. Пополнять баланс (`Deposit`).
3. Снимать средства (`Withdraw`) с проверкой остатка.
4. Замораживать/размораживать счет (блокировка операций).
5. Просматривать информацию о владельце и балансе.
