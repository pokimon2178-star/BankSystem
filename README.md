# BankSystem (Golang Edition) 🚀

Простая консольная система банковского счета, переписанная с C# на Go. Проект показывает, как работают структуры, методы и базовое управление состоянием в Golang.

## 📁 Структура проекта

Я разделил код на логические части, чтобы в нем было легче ориентироваться:

* **types.go** — Здесь лежат «чертежи» нашего счета (структура `BankAccount`) и константы статусов.
* **logic.go** — Вся математика и правила: как создавать счет, как снимать деньги и как проверять заморозку.
* **main.go** — Главный пульт управления. Здесь мы спрашиваем данные у пользователя и запускаем меню.
* **go.mod** — Файл модуля (паспорт проекта), чтобы Go понимал, как всё это запускать вместе.

## 🛠 Функционал

Приложение умеет:
1. Создавать счет с уникальным ID (генерируется случайно).
2. Класть деньги на счет (`Deposit`).
3. Снимать деньги (`Withdraw`) — но только если их хватает!
4. Замораживать счет — тогда никакие операции не пройдут, пока не разморозишь.
5. Показывать инфо о владельце и балансе.
